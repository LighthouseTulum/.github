# Lighthouse Tulum – Tech

Welcome to the Lighthouse Tulum GitHub organization.

This space is used to build and maintain small applications and tools that support the mission, operations, and community life of **Lighthouse Church Tulum**.

## Purpose

Our goal is simple:  
To create practical, lightweight tools that serve the church and help people connect, grow, and engage more effectively.

This may include:
- Internal tools for leadership and operations  
- Event and community management tools  
- Giving or administrative utilities  
- Experimental ideas that support ministry  

## Approach

We aim to keep things:
- **Simple** – no unnecessary complexity  
- **Practical** – built for real use, not theory  
- **Maintainable** – easy for others to understand and contribute  
- **Purpose-driven** – technology always serves ministry, not the other way around  

## Structure

Each repository in this organization represents a standalone project.  
Please include in each repo:
- A clear README  
- Setup instructions  
- Purpose and scope of the tool  

## Contributing

Most contributions will come from a small group of trusted contributors.  
If you are contributing:
- Keep code clean and readable  
- Document anything that is not obvious  
- Prefer clarity over cleverness  

## Tech Stack

We are flexible and pragmatic, but commonly use:
- JavaScript / TypeScript  
- Flutter (for mobile)  
- Firebase or other lightweight backends  

## Final Note

This is not about building software for the sake of building software.  
Everything here exists to support the mission of Lighthouse Church:

Helping people encounter Jesus and walk in His purpose.
